CSS 长度单位参考

这里列出了 CSS 属性 支持的长度单位。除非特别指出，长度单位都可在 Microsoft ? Internet Explorer 3.0 及以后版本中使用。若标有星号(*)，则表明该长度单位可在 Internet Explorer 4.0 及以后版本中使用。

CSS 长度单位表格

相对长度单位 绝对长度单位
em 元素字体的高度。
ex 字母“x”的高度。
px 像素。
% 百分比
in 英寸(1 英寸 = 2.54 厘米)。
cm 厘米。
mm 毫米。
pt 磅(1 磅 = 1/72 英寸)。
pc 皮卡(1 皮卡 = 12 磅)。

样式表属性中的度量和位置是以长度单位给出的。Internet Explorer 支持两种长度单位类型: 相对和绝对。

相对长度单位指定了相对于其它长度属性的长度。相对长度单位在不同的输出设备之间的比例较好，如监视器和打印机。

绝对长度单位指定了绝对的度量，如英寸或厘米。绝对长度单位在输出设备的物理属性未知的情况下非常有用。

当你使用 !DOCTYPE 声明来指定标准兼容模式时，Internet Explorer 6 及以后版本将忽略不遵从 CSS 级别 1(CSS1) 的样式表声明。根据 CSS 1，长度值必须指定单位类型标识符。较早版本的 Internet Explorer 允许你使用某种非法的声明，并将所有没有任何单位类型标识符的数字都假定为以像素为单位，即在字符串后面自动添加“px”后缀。
